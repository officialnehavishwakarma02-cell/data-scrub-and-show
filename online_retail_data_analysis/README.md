# Online Retail Data Analysis
## Project Overview
This project analyzes an online retail dataset to understand customer purchasing behaviour,sales trends and business insights
 using python
 The goal is to clean the dataset and perform exploratory data analysis(EDA) to extract meaningful insights that can help improve
 business decisions

 ## Data Cleaning Steps
 -Hnadled missing values
 -converted date columns to proper datetime format
 -Removed invalid values
 -Created new features:total=quantity*unit price

 ##Exploratory Data Analysis
 - Top ten customers
 - Monthly revenue over time
 - Weekly revenue over time
 - Sales by country
 - -Analyzed repeat customer behaviour

## Technologies used
- python
- pandas
- matplotlib
- seaborn
